# HalluZig: Hallucination Detection using Zigzag Persistence

This repository contains the code used in the paper [HalluZig: Hallucination Detection using Zigzag Persistence](https://arxiv.org/pdf/2601.01552). The paper is accepted at EACL 2026 Main Conference.
